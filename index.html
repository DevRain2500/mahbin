Login & Registration:

Type /register [password] without the []'s to create your account.
then
Type /login to login.

Key Bindings:

  • [B] - Backpack
  • [L] - Lock/Unlock Vehicles
  • [L-Alt] - Interaction Key
  • [T] - Chat

Text Commands:

  • /911 - Places a 911 call. Type this, and then input your text in the popup that follows.
  • /text ##### - Sends a text message to a number. Type /text, followed by they phone number (ie: /text 12345) and then input your text in the popup that follows.
